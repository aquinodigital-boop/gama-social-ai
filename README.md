# Laboratório de Conteúdo

## Descrição
Gerador de conteúdo B2B para a Labor Atacadista.
Utiliza Inteligência Artificial para criar posts de:
- Produtos (com dados técnicos)
- Categorias (Mix Completo)
- Institucional (Logística e Parceria)
